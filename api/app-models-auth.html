queryUsers() public method

A function to query for users to migrate.

This function returns nothing, and the $migrateUsers array should be set in the format seen here $migrateUsers.

where 'username' and 'id' are the username and id from the local database and 'identifier' is the user identifier in from the authentication method. Identifier may also be "NULL", which will set the identifier in the database to null (this is for a migration of a user back to the local authentication method).
